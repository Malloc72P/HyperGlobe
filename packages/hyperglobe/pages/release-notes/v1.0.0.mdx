# HyperGlobe v1.0.0

### 개요

- React와 Three.js 기반 3D 맵차트 라이브러리, [HyperGlobe](https://hyperglobe.malloc72p.com/?path=/docs/components-hyperglobe--docs)의 첫 번째 공식 릴리즈입니다.
- 웹 환경에서 인터랙티브한 3D 지구본과 지리 데이터를 시각화하는 데 최적화되어 있습니다.

### 주요 기능

- [RegionFeature](https://hyperglobe.malloc72p.com/?path=/docs/components-regionfeature--docs): 지구본 위에 국가 및 지역 경계를 표시하는 기능입니다.
- [MarkerFeature](https://hyperglobe.malloc72p.com/?path=/docs/components-markerfeature--docs): 특정 위치에 마커를 추가하여 중요한 지점을 강조할 수 있습니다.
- [RouteFeature](https://hyperglobe.malloc72p.com/?path=/docs/components-routefeature--docs): 경로 데이터를 시각적으로 표현하는 기능입니다.
- [Graticule](https://hyperglobe.malloc72p.com/?path=/docs/components-graticule--docs): 위도 및 경도 그리드를 표시하여 지리적 참조를 제공합니다.
- [Tooltip](https://hyperglobe.malloc72p.com/?path=/docs/components-tooltip--docs): RegionFeature에 툴팁을 추가하여 사용자 상호작용을 향상시킵니다.
- [ColorScale](https://hyperglobe.malloc72p.com/?path=/docs/components-colorscale--docs): 데이터 값에 따라 RegionFeature에 색상을 매핑하는 기능입니다.
