### 개요

`RouteFeature`는 지구본 위 두 지점 사이의 경로를 3D 라인으로 시각화하는 컴포넌트입니다.
대권항로(Great Circle)를 기반으로 경로를 계산하며, `maxHeight` 속성을 통해 포물선 형태의 높이를 조절할 수 있습니다.

주로 `HyperGlobe` 컴포넌트의 `routes` prop을 통해 데이터 객체 형태로 전달하여 사용합니다.
