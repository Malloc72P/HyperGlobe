# HyperGlobe 컴포넌트

- HyperGlobe 컴포넌트는 3D 지구본을 렌더링하는 React 컴포넌트입니다.
- 다양한 속성을 통해 지구본의 외관과 동작을 커스터마이징할 수 있습니다.

<Canvas of={HyperGlobeStory.Overview} />

# import

```tsx
import { HyperGlobe } from '@malloc72p/hyperglobe';
```
