### 개요

`MarkerFeature`는 지구본 위의 특정 좌표에 아이콘과 라벨을 표시하는 컴포넌트입니다.
HTML 요소를 사용하여 렌더링되므로 항상 카메라를 향하며(Billboard), 지구본의 회전에 따라 위치가 업데이트됩니다.
지구본의 반대편으로 넘어갈 경우 자동으로 숨겨집니다.

주로 `HyperGlobe` 컴포넌트의 `markers` prop을 통해 데이터 객체 형태로 전달하여 사용합니다.
